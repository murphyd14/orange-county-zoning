Data Sources

This application uses zoning and future land use data from Orange County, Florida's public GIS services.

Data Processing

  • Geometry simplified for web performance
  • Areas calculated using equal-area projection
  • Zoning codes classified into functional groups
  • Data optimized for free hosting platforms

Technical Details

  • Built with MapLibre GL JS for vector tile support
  • Uses PMTiles format for efficient data delivery
  • Client-side filtering and analysis
  • Export functionality for filtered data

Disclaimers

This is a demonstration application. For official zoning information, please consult Orange County's official GIS portal.
